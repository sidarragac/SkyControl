# SkyControl

## Pre-requisites

- **Node**: Version 22.12.0 or higher (https://nodejs.org/es)
- **NPM**: Version 10.9.0 or higher (https://www.npmjs.com/)
- **Git**

---

## Initial Configuration

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sidarragac/SkyControl.git skycontrol
   cd skycontrol
   ```

Note: *The project is in a Single Page Application phase done in VUE3. Follow these instructions to setup and run the project correctly:*

2. **Open the 'frontend' folder**:
   ```bash
   cd frontend
   ```

3. **Install depencencies**:
   ```bash
   npm intall
   ```

---

## Run the application

```bash
npm run dev
```
