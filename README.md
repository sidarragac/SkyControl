# SkyControl

## Pre-requisites

- **Node**: Version 22.12.0 or higher (https://nodejs.org/es)
- **NPM**: Version 10.9.0 or higher (https://www.npmjs.com/)
- **Git**

---

## Initial Configuration

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sidarragac/SkyControl.git skycontrol
   cd skycontrol
   ```

Note: *The project is in a Single Page Application phase done in VUE3. Follow these instructions to setup and run the project correctly:*

2. **Open the 'frontend' folder**:
   ```bash
   cd frontend
   ```

3. **Install depencencies**:
   ```bash
   npm intall
   ```

4. **Open the 'backend' folder**:
   ```bash
   cd ..
   cd backend
   ```

5. **Install depencencies**:
   ```bash
   npm intall
   ```

---

## Run the application
**Open the 'frontend' folder**:
```bash
cd frontend
```
**Run the 'frontend'**:
```bash
npm run dev
```

**Open the 'backend' folder**:
```bash
cd ..
cd frontend
```
**Run the 'backend'**:
```bash
npm run start:dev
```
